# Description

Ce projet est une application simple réalisée avec Hibernate, un framework Java utilisé pour la persistance des données et le mapping objet-relationnel (ORM). Hibernate permet de mapper les classes Java avec les tables d’une base de données relationnelle afin de simplifier les opérations de stockage et de récupération des données.

L'objectif de ce projet est de comprendre comment utiliser Hibernate pour gérer les entités, les sessions et les opérations CRUD (Create, Read, Update, Delete) dans une application Java.

Ce projet est réalisé dans un cadre pédagogique pour pratiquer l’utilisation de Hibernate ORM avec Java et Maven.

## Technologies utilisées

**Hibernate ORM**

**Maven**

**MySQL**
